# SEO – Local SEO

Optimierung für **lokale Suchanfragen** ("Zahnarzt Berlin", "Pizza liefern Leipzig").

## Die 3 Säulen

| Säule | Was | Tool |
|-------|-----|------|
| **Google Business Profile** | Firmeneintrag bei Google Maps | Direkt |
| **Lokale Keywords** | "Rechtsanwalt + Stadt" | Ahrefs |
| **Bewertungen** | Sterne + Rezensionen | Manuell |

## Schema für Local SEO

```json
{
  "@type": "LocalBusiness",
  "name": "Firmenname",
  "address": { "streetAddress": "...", "postalCode": "...", "addressLocality": "..." },
  "telephone": "+49...",
  "openingHours": "Mo-Fr 09:00-18:00"
}
```
