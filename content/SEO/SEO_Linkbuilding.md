# SEO – Linkbuilding & Backlinks

Ein Link von einer **anderen Website** auf deine. Google wertet das als "Vertrauensbeweis".

## Qualität vor Quantität

| Merkmal | Warum wichtig |
|---------|---------------|
| **Domain-Rating** der Quelle | Höherer DR = mehr Link-Power |
| **Thematische Relevanz** | Link aus ähnlicher Branche zählt stärker |
| **Anchor-Text** | Natürlich halten, nicht "Geld-Keywords" |
| **Dofollow vs. Nofollow** | Nur Dofollow gibt Ranking-Power |
| **Platzierung** | Link im Content > Footer/Sidebar |

## Unser Tool-Einsatz

- **Ahrefs Backlink-Checker**: Bestehende Links analysieren
- **Ahrefs Content-Gap**: Finden, wo Wettbewerber Links haben
- **Disavow-File**: Toxische Links entwerten (Gold)

## Toxische Links erkennen

| Signal | Aktion |
|--------|--------|
| DR < 10 | Prüfen, ggf. disavowen |
| Spam-Ankertexte | Disavowen |
| Link von irrelevanten Nischen | Ignorieren/Disavowen |
| Plötzlicher Link-Anstieg | Prüfen (Negative SEO?) |
